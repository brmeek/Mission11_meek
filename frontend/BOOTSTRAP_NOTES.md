# Bootstrap features used (for Learning Suite comment)

Use this exact note in your Learning Suite submission comment so TAs can quickly verify the "not covered in videos" Bootstrap requirement:

1. **Offcanvas component via data attributes** in `src/App.tsx`
   - `data-bs-toggle="offcanvas"`
   - `data-bs-target="#cartPreview"`
   - `data-bs-dismiss="offcanvas"`
2. **Bootstrap theme data attribute** in `src/App.tsx`
   - `data-bs-theme="dark"` on the table header.

These are in addition to Bootstrap Grid layout (`row`, `col-lg-3`, `col-lg-9`).
